== Description ==

This plugin provides a widget to render WordCamp support badges.  It offers the images shown at <a href="http://2014.annarbor.wordcamp.org/2014/08/19/wordcamp-ann-arbor-badges/">http://2014.annarbor.wordcamp.org/2014/08/19/wordcamp-ann-arbor-badges/</a>

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
